# Football Quiz

Игра-викторина о футболе.

## Планы по разработке (Roadmap)

### Google Sign-In (Профессиональная реализация)
Интеграция авторизации через Google для сохранения прогресса и статистики пользователя.

- [ ] **Настройка Firebase:**
    - Подключить Firebase Auth в консоли.
    - Включить метод входа "Google".
    - Добавить SHA-1 ключ проекта.
- [ ] **Зависимости:**
    - Добавить `play-services-auth` и `firebase-auth-ktx` в `build.gradle`.
- [ ] **Архитектура:**
    - Создать `AuthManager.kt` (Singleton) для управления сессиями.
- [ ] **UI/UX:**
    - Создать `ProfileFragment` для отображения данных пользователя.
    - Добавить иконку профиля на `StartPageFragment`.
    - Реализовать логику входа/выхода и синхронизации очков.
